# SDGP-Steel-Beam-Cutting-Optimisation-System

Overview

This project solves the One-Dimensional Cutting Stock Problem (1D-CSP) to minimise material waste when cutting beams. It uses a Python optimisation algorithm with an Electron dashboard for visualisation.

Features
Global optimisation (exhaustive search)
Supports beam sizes: 6000mm, 8000mm, 13000mm
Minimises waste and number of beams
Interactive dashboard with visualisation
CSV export
Runs fully offline
